
### Data Types

### Event
- ID
- Description
- category_id - FK
- Due Date
- Location?
- User_ID - FK

### Category
- ID
- Name
- Type
- T1 - Threshold 1 range (# weeks, # months, # days, etc.)
- T2 - time until 2 range
- T3 - time until 3 range

### User
- ID
- Name (first and last)
- Username
- Password - encrypt

### Settings - Session
- user_id - FK
- view_type
